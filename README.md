# MapperInteractive_new

## How to run

```bash
git clone git@github.com:zhou325/MapperInteractive_new.git
cd MapperInteractive_new
python3 run.py
```

You can view the page at http://0.0.0.0:8080/ (If possible, please use Chrome).

Plugins
---------


Prebuilt Blocks
===================


Plugins define graph behaviors.

Currently implemented graph plugins include

- :code:`Draggable`
- :code:`ForceSimulation`
- :code:`Labeled`
- :code:`MarkerEnd`
- :code:`Popover`

Read more about how to define your own plugin in :ref:`Plugins`.

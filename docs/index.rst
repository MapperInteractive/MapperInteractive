MapperCore
-------------


The Mapper algorithm is a method for topological data analysis invented by Gurjeet Singh, Facundo MÃ©moli and Gunnar Carlsson [^1]. MapperCore is a web-based visualization of Mapper algorithm. It is open source software and is released under the MIT License.

Please find the source `on Github <https://github.com/mappercore/mappercore>`_.


Installation
==============

MapperCore can be installed from PyPI by:

..

    pip install mappercore==1.2.6
